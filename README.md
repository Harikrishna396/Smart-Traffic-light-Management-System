# Smart-Traffic-light-Management-System
PROTOTYPE: https://github.com/Harikrishna396/Smart-Traffic-light-Management-System/blob/main/project%20prototype.jpeg

Smart Traffic Light System developed using Arduino and IR sensors to control traffic signals based on real-time vehicle density. The system automatically adjusts signal timings by detecting traffic on different lanes, helping reduce congestion, waiting time, and improving traffic flow efficiency at road intersections.
